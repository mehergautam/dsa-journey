# Queue – Data Structure (Day 9)

## What is Queue?

Queue is a linear data structure that follows:

FIFO (First In First Out)

## Basic Operations

- Enqueue (Insert element)
- Dequeue (Remove element)
- Front (Get first element)
- isEmpty
- isFull (for array implementation)

Time Complexity:
- Enqueue → O(1)
- Dequeue → O(1)
- Front → O(1)
