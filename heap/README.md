# Heap / Priority Queue

Heap is a complete binary tree that satisfies the heap property.
Types of Heap:
- Min Heap
- Max Heap

Heaps are commonly used for problems involving:
- Finding Kth smallest or largest elements
- Merging sorted arrays or lists
- Priority-based processing

## Implemented Problems
1. Kth Largest Element using Min Heap
2. Merge K Sorted Arrays using Priority Queue
3. top_k_frequent_elements.cpp
4. k_closest_numbers.cpp
5. k_closest_points_to_origin.cpp
6. connect_ropes_min_cost.cpp

## Applications

- Scheduling algorithms
- Graph algorithms (Dijkstra, Prim)
- Data stream processing
