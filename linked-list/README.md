# Linked List – Singly Linked List Implementation

This section covers the fundamental implementation of a Singly Linked List in C++ as part of my structured DSA journey.

## 📌 What is a Linked List?

A Linked List is a linear data structure where elements are stored in nodes.  
Each node contains:
- Data
- Pointer to the next node

Unlike arrays, linked lists use dynamic memory allocation.
