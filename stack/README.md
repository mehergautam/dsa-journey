# Stack – Data Structure (Day 8)

## What is Stack?

Stack is a linear data structure that follows:

LIFO (Last In First Out)

## Basic Operations

- Push (Insert element)
- Pop (Remove top element)
- Peek (View top element)
- isEmpty (Check if stack is empty)

Time Complexity:
- Push → O(1)
- Pop → O(1)
- Peek → O(1)
