## Level Order Traversal

Level Order Traversal visits nodes level by level.
It uses Queue (FIFO principle).

Time Complexity: O(n)

## Height of Tree

Height = Maximum number of levels in the tree.

## Count Nodes

Counts total number of nodes using recursion.
